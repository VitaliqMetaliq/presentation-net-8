# presentation-net-8

### Проект для презентации
Разделен на несколько микросервисов:
Background - HostedService, который обновляет таблицу валют <br/>
UserService - методы Login, Register - logout нету, вместо этого - ограничение жизни токена в конфиге <br/>
Finance - получение валют по юзеру, добавление валют для текущего авторизованного юзера, а также grpc метод для получения валют по юзеру <br/>
Gateway - валидация токена и проксирование, также выставлен контроллер для того, чтобы можно было дернуть grpc метод, который реализован в Finance <br/>
DbMigrator - консольное приложение, выполняет миграцию и останавливается <br/>
Shared.* - все, что повсеместно используется во всех (или части) проектах <br/>

При реализации старался использовать Clean Architecture, настроил запуск в контейнере (docker compose -f docker-compose.yml up -d)
для всех сервисов, а также выложил postman_collection которым пользовался, для удобства.
